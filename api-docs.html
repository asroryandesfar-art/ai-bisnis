POST /auth/register public

Daftar organisasi baru + user owner pertama. Otomatis mulai trial 14 hari gratis.

Request body

FieldTypeKeterangan
org_namestring*Nama perusahaan kamu
emailstring*Email untuk login
passwordstring*Min 8 karakter
full_namestringNama lengkap (opsional)

Contoh request

POST /auth/register
Content-Type: application/json

{
  "org_name":  "TokoPintar",
  "email":     "rizky@tokopintar.co",
  "password":  "p4ssw0rd!",
  "full_name": "Rizky Setiawan"
}

Response 201

{
  "token":       "eyJhbGci...",
  "org_id":      "a1b2c3d4-...",
  "trial_ends":  "2025-09-04T10:00:00Z"
}
POST /auth/login public

Login dan dapatkan JWT token. Token berlaku 7 hari. Sertakan di header semua request yang memerlukan auth.

Contoh request

POST /auth/login

{ "email": "rizky@tokopintar.co", "password": "p4ssw0rd!" }

Cara pakai token

Authorization: Bearer eyJhbGci...

Response 200

{ "token": "eyJhbGci..." }

POST /bots/{bot_id}/documents auth required

Upload dokumen untuk knowledge base bot. Format: PDF, DOCX, TXT. Diproses async — status bisa dicek via list documents.

Request (multipart/form-data)

POST /bots/uuid/documents
Content-Type: multipart/form-data

file: [binary — FAQ.pdf]

Contoh curl

curl -X POST \
  https://api.botnesia.id/v1/bots/UUID/documents \
  -H "Authorization: Bearer TOKEN" \
  -F "file=@/path/to/FAQ.pdf"

Response 201

{ "doc_id": "uuid", "status": "processing" }

POST /chat/{bot_id} public

Core chat endpoint — dipanggil oleh iframe widget. Otomatis RAG dari knowledge base + kirim ke Claude API.

Request body

FieldTypeKeterangan
messagestring*Pesan dari end-user. Max 2000 karakter
session_idstringUUID conversation yang sedang aktif. Kosong = mulai baru
user_metaobjectData user dari ChatbotWidget.identify(). Scale tier.

Contoh request (Scale tier dengan identity)

  "user_meta": {
    "userId":     "user_9182",
    "name":       "Budi Santoso",
    "plan":       "premium",
    "orderCount": 14
  }
}

Response 200

{
  "answer":     "Halo Budi! Pesanan #TP-88291...",
  "session_id": "conv-uuid-...",
  "latency_ms": 842
}

POST /webhooks auth required · Growth+

Daftarkan URL untuk menerima event real-time. Tiap event dikirim dengan HMAC signature untuk verifikasi.

Request body

{
  "url":    "https://kamu.com/webhook",
  "events": [
    "conversation.started",
    "handoff.needed",
    "rating.submitted"
  ]
}

Response 201

{
  "webhook_id": "uuid",
  "secret":     "abc123...",
  "note":       "Simpan secret ini — tidak akan ditampilkan lagi."
}

Verifikasi signature di server kamu

# Python
import hmac, hashlib
sig = hmac.new(
  secret.encode(),
  request.body,
  hashlib.sha256
).hexdigest()
assert sig == request.headers["X-BotNesia-Signature"]
